Alcatraz使用「管理Xcode插件」(插件推荐)

Alcatraz使用「管理Xcode插件」(插件推荐)

帮助你使用可视化的方式管理Xcode插件


Alcatraz


Alcatraz安装:

下载地址: GitHub - alcatraz/Alcatraz: Package manager for Xcode

1.下载后运行Command+R

2.完全退出Xcode重新打开

3.选择Load Bundles(同意加载工具包)


同意加载工具包

4.此时Window中会多了一个Package Manager,打开即可管理Xcode插件了



5.在Search搜索栏搜索自己要想要安装的插件,点击INSTALL安装,安装完成后重启Xcode即可

记住同意加载工具包哦! (选择Load Bundles)


Alcatraz插件管理


6.安装后可在Installed标签下查看,安装的插件是否成功,还可以很方便的卸载


Alcatraz插件管理


Xcode中常用插件推荐:

1.VVDocumenter

下载地址: GitHub - onevcat/VVDocumenter-Xcode: Xcode plug-in which helps you write documentation comment easier

喵神出品,只需输入''///''即可,自动生成规范化的代码注释!


VVDocumenter


2.ESJsonFormat

下载地址: GitHub - EnjoySR/ESJsonFormat-Xcode: 将JSON格式化输出为模型的属性

将JSON格式化数据自动转换为Model模型的属性,支持层级嵌套JSON数据!


ESJsonFormat


3.XAlign

下载地址: GitHub - qfish/XAlign: An amazing Xcode plugin to align regular code. it can align Xnything in any way you want.

用来对齐常规代码,也可以自定义对齐模式!


XAlign


4.FKConsole

下载地址: GitHub - Forkong/FKConsole: FKConsole is a plugin for Xcode to adjust console display(about Chinese).

在控制台直接将UTF8编码转化为中文!


FKConsole


5.KSImageNamed

下载地址: GitHub - ksuther/KSImageNamed-Xcode: Xcode plug-in that provides autocomplete for imageNamed: calls

使用[UIImage imageNamed:@"xxx"]时会自动扫描项目中图片文件!


KSImageNamed


6.ColorSense

下载地址: GitHub - omz/ColorSense-for-Xcode: Plugin for Xcode to make working with colors more visual

颜色转换!


7.CocoaPods for Xcode

下载地址:  github.com/kattrali/cocoapods-xcode-plugin

可视化的使用CocoaPods做iOS程序的依赖管理


CocoaPods for Xcode


8.ESTranslate

下载地址: GitHub - EnjoySR/ESTranslate-Xcode: Xcode代码单词快速翻译

快速翻译Xcode中代码单词以及句子!


ESTranslate

9.Injection Plugin for Xcode

下载地址: GitHub - johnno1962/injectionforxcode: Runtime Code Injection for Objective-C & Swift

实时在模拟器或实机上看到效果而不需要重启应用


Injection Plugin for Xcode
