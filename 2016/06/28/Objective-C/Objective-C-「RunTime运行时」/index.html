RunTime简称运行时。就是系统在运行的时候的一些机制，其中最主要的是消息机制。
RunTime是一套比较底层的纯C语言API，属于1个C语言库，包含了很多底层的C语言API。
在我们平时编写的OC代码中，程序运行过程时，其实最终都是转成了runtime的C语言代码。
