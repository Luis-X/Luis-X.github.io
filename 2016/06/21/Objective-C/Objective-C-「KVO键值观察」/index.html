Objective-C-「KVO键值观察」

发表于 2016-06-21   |   分类于 Objective-C   |  
  • Key Value Observing(键-值-观察)
  • 被观察者和观察者同时实现一个协议: NSKeyValueObserving
  • 源于设计模式中的观察者模式。
  • 指定被观察对象的属性被修改后，KVO就会自动通知相应的观察者。
    注意:不要忘记解除注册，否则会导致资源泄露
